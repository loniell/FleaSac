To personalise portfolio and blog pages, follow next steps

Every list item from projects list has a class attribute with the name of category ( Ex: class="webdesign", class="webdevelopment" ...) and data-id ( here you have to put an increment ID number ). When a user press on a .sidemenu link, script will analyse each .projects list element and it will hide or show, if the list "class" attribute is equal with "data-value" from clicked link.

To change categories, first you have to change the data-value attribute value, for each ".sidemenu list" link with the new category (at line 284). After that, you have set the category for each list element from .projects list (Ex <li data-id="ID" class="CATEGORY_NAME">).

The sorting system works automatically by reading some elements from .projects list items: H3 element for sort by title , data-id attribute for sort by ID, and div.price->span number for sort by price.
IMPORTANT - If you want to change this sort type, for example by sales or views, don't change the value "price" from class attribute, change olny the PRICE text. EXAMPLE: <div class="price"> SALES:  <span>10</span> </div>
Don't forget to change the PRICE text from portfolio_sort list, at line 303 and leave data-value="price" unchanged.

These steps are available for BLOG page too.

B) CSS Files and Structure - top

The CSS file contains all of the specific stylings for the page with a reset style. Many browser interpret the default behavior of html elements differently. By using a general reset CSS file, we can work round this.

The main CSS file is well documented and named, so you will not have problems to change or add something. CSS file structure looks like this :

	 

/* RESET & BASIC */

/* LOGO & MENU */

/* PAGE SLIDER */

/* PAGE */

/* SUBPAGE */

/* Contact form */

/* The Nivo Slider styles */

/* LIST TYPE */

/* PORTFOLIO & BLOG */

/* BUTTONS */

/* BODY.light STYLE */

/* FANCYBOX */

If you would like to edit a specific section of the site, simply find the appropriate label in the CSS file, and then scroll down until you find the appropriate style that needs to be edited.

#logo-menu {
	change styles here
}

If you want to paginate a list (like that from About Us page) you have to use this code
$("LIST").paginate({ limit: ELEMENTS_PER_PAGE, content: 'li' }); You can find the example at line 140 in custom.js file.
At line 144 from same file you can set the settings for Contact form (More info at http://docs.jquery.com/Plugins/validation).

D) PSD Files - top

I've included 10 PSD files with this theme.
Let's say you want to change the transparency or color for page holder (dark version).
  1. Open content.psd file, and edit shape1 style, after that right click (on the layer), and click on copy layer style
  2. Open content-top.psd, content-middle.psd, content-bottom.psd -> riht click on the layer and press on paste layer style (do this for each).
  3. Overwrite to the existing .png files on images folder, and you are done
