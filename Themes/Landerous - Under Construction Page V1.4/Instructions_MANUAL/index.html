A) HTML Structure - top

This theme is a width fixed layout. All page parts which is centered is wrapped with a css class "wrapper blockCenter".

Header Content

Header is nested in div with class of "header". Inside this section there is only a logo.

Rocket Spacer

This part of template is used just to separate logo and content bellow.

Site Introduction Content

Site introduction content have two parts. One is a text which is displayed on the left side ("WE ARE CURRENTLY WORKING ON..."), and on the right side, there is a landing rocket image. Introduction text is nested in a div with a class of "info". This is also a wrapper for a div with a class of "time" which is used to hold unorder list where is counter displayed. Also there is nested an image "Time to launch".

Sing Up

Sing Up content is nested in div with a class of "lightBlue".

About Us, Our Twitts, Contact Us

These parts of template is nested in a div with a class of "darkBlue". Each of these parts (about us, twitts, contact) have their own wrapper which is a div with a class called just like a text abowe "about" "ourTwitts" "contactUs".

Footer Content

Footer is nested in div with a class of "footer". Footer contains two main parts, basic copyright notes which is stored in div with class of "copyright", and menu section which is nested in div with class of "floatRight".

B) CSS Files and Structure - top

I'm using a few CSS files in this theme. These files are stored in folder named "css".

clear.css

This file contains specific stylings to reset all browser default styles, so we can avoid different rendering.

common.css

This file contains specific style sheets which can be used in some other project also. For example for fixing content left or right, making text bold, italic, underline etc.

main.css

This file contains specific stylings for this site.

If you have any additional question about how to customize some part on template, fill free to contact us.

C) JavaScript - top

This theme imports these Javascript files.

  1. jQuery
  2. myHint
  3. main
  1. jQuery is a JavaScript library that greatly reduces the amount of code that you must write.
  2. myHint is our jQuery plugin, which is used for displaying default hint in input elements, and in text area.
    This is a example how you can use this plugin: 
    $('#your_jQuery_selector').attachHint('Here you can put default value for wanted element');
  3. main, as this name already says is our main JavaScript file. This script contains code for countdown, for sending email notification via ajax when someone sing up on page, for sending contact message to site owner, validating user input, displaying twitts, and binding all these events together on page.

D) PSD Files - top

Landerous is pretty clean and minimal template and you basically don't need advanced Photoshop skills to edit this theme. The most things you can easilly change via CSS. Anyway, you'll need .PSD files for background, logo and social icons change. In the "Landerous_PSD" folder, you'll find two .PSD files.

  1. Landerous.PSD
  2. Landerous - Social Icons.PSD (stumble, vimeo, flickr, linkedin, facebook, skype, twitter, rss icons included)

If you want to change background, color, you'll need to change two JPEG's. Let's take "Landerous_LightBlue" theme as example from here. So, in the folder "Landerous_LightBlue\images" you'll find two JPEG's you need to change. First one is "dark_blue_bg.jpg" and second (lighter) one is "light_blue_bg.jpg". Open both files in the Photoshop and paint them in desired color. After that action, just replace "dark_blue_bg.jpg" and "light_blue_bg.jpg" with new ones that you jst create. And that's how you'll get new background color. Same procedure is for "contact_bg.png" and "subscribe_bg.png". If you want to change the logo, just replade current one "Landerous_LightBlue\images\logo.png" with your own.

If you don't like the rocket graphic, you can change that with your own photo or illustration as well.

F) PHP Code Explanation - top

PHP script is used for sending email from page.
In "CONFIG.php" you will need to change only this parts:

//enter site start date
$start_date = "2012/09/26"; //<-- Your site start date must be in this format yyyy/mm/dd

//enter your email here
$email = 'drythemes@gmail.com'; //<<-- change this value here only to your address.
//enter your twitter name here
$twitter_name = "drythemes"; //<-- Twitter name you want to follow (show tweets) on page
//enter here email address where you want to receive notifications about new subscribers
$singUpEmail = "no-reply@drythemes.com"; //<<-- Email address for sending you subscribe notification
//enter here the subject of the notification email that you want to receive
$singUpNotificationSubject = "New site subscriber!"; //<<-- Email subscribe notification subject

One notice: this will be functional only on your server, php file can not be executed on your local computer.
